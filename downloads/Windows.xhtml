GIMP for Windows

The Windows installer packages by Jernej Simončič are now distributed from the GIMP FTP server (ftp.gimp.org) and its mirror servers. The ads on the previously used site have tricked too many users into downloading wrong apps or installers that have been tampered with, so we decided to cut that out.

The installer contains both 32 bit and 64 bit versions of GIMP, and will install the appropriate one.

  • Download GIMP 2.8.10 via HTTP or FTP – Installer for Windows XP SP3 or later
