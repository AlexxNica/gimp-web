Source for version 2.4 (Stable)

The GIMP releases available from gimp.org and its mirrors contain the source code and have to be compiled in order to be installed on your system.

GIMP 2.4.0 is now available at ftp://ftp.gimp.org/pub/gimp/v2.4/. You may want to read the Release Notes for GIMP 2.4.

Development version

GIMP 2.5.0 is a snapshot of current development towards GIMP 2.6. The source code is available from ftp://ftp.gimp.org/pub/gimp/v2.5/. Please make sure you read the Release Notes for GIMP 2.5.

A detailed list of changes in the development branch is available at developer.gimp.org.
