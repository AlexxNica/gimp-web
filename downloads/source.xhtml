Source for version 2.4 (Stable)

GIMP releases available from gimp.org and its mirrors contain the source code and have to be compiled in order to be installed on your system.

GIMP 2.4.3 is now available at ftp://ftp.gimp.org/pub/gimp/v2.4/. You may want to read the Release Notes for GIMP 2.4.
