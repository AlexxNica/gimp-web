GIMP for Mac OS

the GIMP team doesn't provide official Mac binaries. You can, however, install GIMP easily on the Mac using the universal binary provided by the gimp.app project.

Running GIMP on the Mac OS X requires Apple's X11 environment. It is included on the "Optional Installs" package on the OS X install disk.

  • Download Gimp.app – for Mac OS X 10.3 and above
