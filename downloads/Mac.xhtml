GIMP for Mac OS X

There are two ways to run GIMP on OSX: natively (running directly on top of OSX), and using X11 (requires Apple's X11 environment):

Native Build

This is a GIMP 2.8 DMG installer. Just open the downloaded DMG and drag "GIMP.app" into your "Applications" folder.

  • Download gimp-2.8.2-dmg-2.dmg

X11 Build

These packages install just as easily but require X11. Lisanet's builds include some more useful additional plug-ins. Get them from GIMP on OS X.

  • Download GIMP 2.8

Macports

Disclaimer: we have not investigated if it is possible to build GIMP 2.8 from Macports

An easy way to compile and install GIMP and other great Free software on your Mac is by using Macports. The installer allows you to choose from a large directory of packages. To install gimp using Macports, you simply do sudo port install gimp once you have Macports installed.

  • Download Macports.

Fink

Disclaimer: we have not investigated if it is possible to build GIMP 2.8 from Fink

Fink is a package repository that offer mostly precompiled binaries. It provides the apt-get command known to e.g. Debian and Ubuntu users, and installing GIMP is as easy as sudo apt-get install gimp once you have installed the Fink installer.
If there's no binary package, then fink install gimp will compile GIMP from source.

  • Download Fink.
