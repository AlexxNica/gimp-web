GIMP for Mac OS X

The GIMP team doesn't provide official Mac binaries. You can, however, install GIMP 2.4 easily on the Mac using the Intel binary provided by the gimp.app project.

Running GIMP on the Mac OS X requires Apple's X11 environment. It is included on the "Optional Installs" package on the OS X install disk.

  • Download Gimp.app 2.4 – Intel binary for Mac OS X 10.3 and above
  • Download Gimp.app 2.2 – universal binary for Mac OS X 10.3 and above

Fink & Macports

Another easy way to install GIMP and other great Free software on your mac is downloading either Fink or Macports. The installers then allow you to choose from a large directory of packages. To install gimp using fink, you simply do sudo apt-get install gimp once you have Fink installed. Similarly on Macports you do sudo port install gimp.

  • Download Macports.
  • Download Fink.
