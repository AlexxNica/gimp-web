Tips and Tricks for user_install

This is the dialog you see when you open The GIMP for the first time.

There is probably no problem to just agree to all of the defaults. However, The GIMP installation can use some tweaking here and there to make it appear and behave perfectly for you. Some of these options are set during this dialog.

Welcome

Since this window mentions the GNU General Public License you know it is truly a Welcome dialog you are entering into. ;) (screenshot).

Also, note the "Continue" button. The GIMP does not even ask that you agree to it, merely to continue. Feel free to press the continue button.

User Installation Log

This window shows you the files that The GIMP will make. It will have some complaints if you told it to install some place that it didn't have permission to be. There is a scroll bar to see all the things The GIMP has for you. (screenshot).
