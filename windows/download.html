Download GIMP for Windows

WARNING: none of the GIMPs on these page are official. You take your own chances. The only official GIMPs are those available in source format (not executable) at ftp.gimp.org.

There is no official version of the GIMP for Windows, and www.gimp.org does not qualify any as such. However, Tor Lillqvist has been making builds for Windows for awhile now, and most Windows' versions of the GIMP are based on his builds.

Go to Tor's download page in order to download his sources or his executables (programs that can be run).

The easiest GIMP download can probably be found at Jernej Simoncic's download page.

Whichever download page you choose, please follow the instructions on those pages closely!

Should there be easier installations?

Very probably there should. GIMP being free software and all, you are entirely welcome to make such an easy installation and report to us when you have made it.

Other GIMPs for Windows

We have seen a few reports of people building GIMP under CygWin (which is a sort of Unix that runs in a console on the Windows desktop). If anyone could tell us some more about this, we could report it on these pages.
