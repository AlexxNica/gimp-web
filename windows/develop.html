How to help develop the GIMP

GIMP is developed mostly under Linux, but it may be compiled for other systems such as Windows too.

WARNING: none of the GIMPs on these pages are 'official'. You take your own chances. The only official GIMPs are those available in source format (not executable) at ftp.gimp.org.

Programming

Most GIMP coders use the Linux platform, and you'll probably find that things go a lot easier if you do also.

If you're not a programmer, you can help the GIMP grow in many other ways. Help

  • write documentation,
  • translate the GIMP,
  • translate the documentation,
  • write tutorials,
  • make artwork used in or with the GIMP,
  • make sure that people know you used the GIMP for your artwork,
  • sell or give away copies of the GIMP, or
  • help others to learn to use the GIMP.

So you see, it does not really matter whether you can program or not. Of course, programmers are extremely valuable to the project, perhaps even more so Windows programmers, but there are many other ways in which you can contribute.

If you decide to 'evangelize', though, try and stay civil. You catch more flies with honey than with that other stuff.
